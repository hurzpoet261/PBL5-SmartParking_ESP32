# 🚀 BẮT ĐẦU TẠI ĐÂY - Smart Parking System

## ⚡ Chạy nhanh (3 bước)

### 1️⃣ Kiểm tra hệ thống
```bash
VERIFY_SETUP.bat
```

### 2️⃣ Tạo database
```sql
-- Mở PostgreSQL và chạy:
CREATE DATABASE smart_parking;
```

### 3️⃣ Chạy setup và start
```bash
# Lần đầu tiên
setup.bat

# Khởi động
START.bat
```
